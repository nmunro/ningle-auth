# ningle-auth 0.2.0

## Author

nmunro

## Description

Ningle-Auth is a user registration and login application, it uses [cu-sith](https://github.com/nmunro/cu-sith/) to manage the authentication, it also uses [envy](https://github.com/fukamachi/envy) and [envy-ningle](https://github.com/nmunro/envy-ningle), please ensure you have included a `:auth-mount-path` in your settings.

The following environmental variables are required:

* APP_ENV
* ENVY_CONFIG_PACKAGE
